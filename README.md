# DecodeBot - Rule Based Chatbot

## Description

DecodeBot is a simple rule-based chatbot developed using Python. It responds to predefined user queries using conditional statements and provides basic conversational interaction.

## Features

* Greeting responses
* AI and Python related responses
* Handles uppercase and lowercase input
* Handles extra spaces in user input
* Exit command to end the conversation

## Technologies Used

* Python

## How to Run

1. Open terminal or command prompt.
2. Navigate to the project folder.
3. Run:

python chatbot.py

## Sample Inputs

* hello
* hi
* how are you
* what is ai
* what is python
* thank you
* good morning
* good night
* goodbye
